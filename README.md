# ⚖️ AI Legal Guide

Welcome to **AI Legal Guide** — a project designed to assist with legal insights using AI.

---

## 🚀 Getting Started

### Step 1: Install Dependencies
```bash
npm i
```
### Step 2: Run Development Server
```bash
npm run dev
```
###  Step 3: Build for Production
```bash
npm run build
```
### Step 4: Start Production Server
```bash
npm start
```
